# OpenParliamentTV-Data-DE-BY
Open Parliament TV Data Repository for the Bavarian state parliament (Bayerischer Landtag)

> **Status:** in development

## Pipeline

This data is generated by the `DE-BY` parliament implementation in
[OpenParliamentTV-Tools](https://github.com/OpenParliamentTV/OpenParliamentTV-Tools):
[`optv/parliaments/DE-BY`](https://github.com/OpenParliamentTV/OpenParliamentTV-Tools/blob/main/optv/parliaments/DE-BY).
